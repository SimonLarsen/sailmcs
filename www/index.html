SailMCS

SailMCS is an inexact algorithm for the multiple maximum common edge subgraph problem.

Download

The source code is available at http://github.com/SimonLarsen/sailmcs.git.

Test data and supplementary material is available here.

Usage

sailmcs -t 600 -o mcs.sif -O alignment.txt graph1.sif graph2.sif graph3.sif

Citation

A. Author, B. Author, and C. Author. A Simulated Annealing Algorithm for Maximum Common Edge Subgraph Detection in Biological Networks. Journal Name, 1(2):34-56, 2015.
