SailMCS

Simulated Annealing Iterated Local search for the Maximum Common Subgraph problem

About SailMCS

SailMCS is a heuristic algorithm for the multi-MCES problem. Using a combination of simulated annealing and iterated local search, it is able to find good alignments and large substructures for multiple, large graphs.

Download

The source code is available at http://github.com/SimonLarsen/sailmcs.git.

Test data and supplementary material is available here.
