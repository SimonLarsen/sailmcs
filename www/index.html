SailMCS

A heuristic algorith for the multiple maximum common subgraph problem

Download

The source code is available at http://github.com/SimonLarsen/sailmcs.git.

Test data and supplementary material is available here.

Compilation

Requirements:

  • Modern compiler supporting C++11 and OpenMP
  • Make
  • CMake >= 2.8
  • Boost >= ?
  • help2man (optional, used for man page generation)

With all requirements installed, run the following commands:

git clone --recursive https://github.com/SimonLarsen/sailmcs.git
cd sailmcs
cmake . && make

Usage

sailmcs -t 600 -o mcs.sif -O alignment.txt graph1.sif graph2.sif graph3.sif
